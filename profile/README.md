# Jobcelis

**Event infrastructure platform for developers** — publish events, configure webhooks with advanced filters, receive real-time deliveries, and schedule recurring jobs with cron expressions.

Built with **Elixir/OTP**, **Phoenix 1.8**, **LiveView**, and **PostgreSQL**.

### Key Features

- **90+ REST endpoints** with JWT and API Key authentication
- **Real-time webhooks** with retries, circuit breaker, and batch delivery
- **Scheduled jobs** with cron expressions and execution history
- **Multi-step pipelines** for event processing workflows
- **Event replay** and dead letter queue management
- **Multi-project** workspaces with team roles (owner/editor/viewer)
- **GDPR compliant** — data export, erasure, consent management
- **Enterprise security** — encryption at rest, MFA/TOTP, anomaly detection, audit log
- **OpenAPI 3.0** with interactive Swagger UI
- **Real-time streaming** via SSE and WebSocket
